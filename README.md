## Versioning

This project uses [Semantic Versioning](https://semver.org/). The version number is maintained in the `package.json` file.

To update the version:
1.  Open `package.json`.
2.  Locate the `"version"` field.
3.  Increment the version number according to SemVer rules (Major.Minor.Patch).
4.  Commit the change.
